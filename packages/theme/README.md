# @yamada-ui/theme

The default theme package for Yamada UI components.

## Installation

```sh
$ pnpm add @yamada-ui/theme
```

or

```sh
$ yarn add @yamada-ui/theme
```

or

```sh
$ npm install @yamada-ui/theme
```
